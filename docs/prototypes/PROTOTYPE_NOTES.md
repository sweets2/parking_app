# Prototype Notes

The project may include four prototype images in this directory:

- prototype-01.png
- prototype-02.png
- prototype-03.png
- prototype-04.png

These images are references for the map/app visual direction.

Use them to guide:
- map-first layout
- Check | Rules navigation
- bottom sheet proportions
- mobile spacing
- desktop responsiveness
- color/status hierarchy
- segment detail presentation
- touch target size

Do not treat the images as a full product spec.

Written feature specs override prototype images.

Do not introduce:
- saved spots
- reminders
- notifications
- push
- background monitoring
- alert mode
