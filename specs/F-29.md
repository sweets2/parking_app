## F-29 Direction Badge Redesign `[DONE]`

Redesigns the compass-direction badges in the street-cleaning popup. Each direction gets a distinct compass-traditional color scheme, and small triangle arrows are added via CSS pseudo-elements to reinforce directionality.

Files modified:

- `app/style.css` — new badge color palette, arrow pseudo-elements, sizing and positioning rules
- `app/map.ts` — simplify `directionBadge()` to emit plain letter badges (no inline triangle character); arrows are rendered by CSS only

---

### `app/map.ts` — modification

`directionBadge(side: string): string`

Returns one of five `<span>` variants based on `side.toLowerCase()`:

| side value | class(es) | letter |
|---|---|---|
| `"north"` | `dir-badge dir-n` | `N` |
| `"south"` | `dir-badge dir-s` | `S` |
| `"east"`  | `dir-badge dir-e` | `E` |
| `"west"`  | `dir-badge dir-w` | `W` |
| anything else | `dir-badge dir-other` | `side.charAt(0).toUpperCase()` |

No inline triangle character — arrows are rendered by CSS `::before` / `::after` pseudo-elements.

---

### `app/style.css` — modifications

**Base badge** (`.dir-badge`):

```css
.dir-badge {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  position: relative;
  padding: 2px 4px;
  border-radius: 8px;
  font-size: 11px;
  font-weight: bold;
  white-space: nowrap;
  flex-shrink: 0;
  min-width: 26px;
}
```

**N/S sizing override** — extra vertical padding to accommodate top/bottom arrows without touching the letter:

```css
.dir-n, .dir-s { padding: 3px 3px; min-width: 22px; }
```

**Arrow pseudo-elements** — absolutely positioned, `font-size: 11px`, `line-height: 1`:

```css
.dir-n::before { content: "▴"; font-size: 11px; line-height: 1; position: absolute; top: -2px;   left: 50%; transform: translateX(-50%); }
.dir-s::after  { content: "▾"; font-size: 11px; line-height: 1; position: absolute; bottom: 0px; left: 50%; transform: translateX(-50%); }
.dir-e::after  { content: "▸"; font-size: 11px; line-height: 1; position: absolute; right: 1px;  top: 50%; transform: translateY(calc(-50% - 1px)); }
.dir-w::before { content: "◂"; font-size: 11px; line-height: 1; position: absolute; left: 1px;   top: 50%; transform: translateY(calc(-50% - 1px)); }
```

**Color palette** — compass-traditional:

```css
.dir-n { background: #fee2e2; color: #dc2626; padding-top: 5px; }
.dir-s { background: #dbeafe; color: #1d4ed8; padding-bottom: 5px; }
.dir-e { background: #dcfce7; color: #15803d; }
.dir-w { background: #fef3c7; color: #b45309; }
.dir-other { background: #f3f4f6; color: #374151; }
```

N has `padding-top: 5px` and S has `padding-bottom: 5px` as per-direction overrides that compensate for asymmetric glyph positioning within the em box.
