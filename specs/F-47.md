# F-47 — Natural Language Time Parser

## What this is

A pure function module that converts plain-English parking queries into a
structured time window. No browser globals, no I/O, no imports beyond types.

```typescript
export interface ParkingQuery {
  startTime: Date;   // window start (inclusive)
  endTime:   Date;   // window end   (inclusive)
  label:     string; // human-readable, e.g. "overnight" — shown in the query bar
}

export function parseQuery(text: string, now: Date): ParkingQuery | null
```

Returns `null` for unrecognized input. Callers show an error message; they do
not fall back silently.

## `shared/query-parser.ts`

### Supported patterns (case-insensitive, leading/trailing whitespace stripped)

| Pattern | `startTime` | `endTime` | `label` |
|---------|-------------|-----------|---------|
| `"overnight"` / `"all night"` / `"tonight"` | 10 pm today (if now < 10 pm), else now | 8 am tomorrow | `"overnight"` |
| `"right now"` / `"now"` | `now` | `now + 1 hour` | `"right now"` |
| `"for N hours"` (N ≥ 1) | `now` | `now + N hours` | `"for N hours"` |
| `"for N hour"` (singular) | `now` | `now + 1 hour` | `"for 1 hour"` |
| `"for N minutes"` / `"for N mins"` / `"for N min"` (N ≥ 1) | `now` | `now + N minutes` | `"for N minutes"` |
| `"until Xam"` / `"until Xpm"` / `"until X am"` / `"until X pm"` (X = 1–12) | `now` | today at X:00 am/pm (tomorrow if that time is already past) | `"until X am/pm"` |
| `"until H:MM am"` / `"until H:MM pm"` | `now` | today at H:MM am/pm (tomorrow if past) | `"until H:MM am/pm"` |
| `"all day"` | start of today (midnight) | end of today (11:59 pm) | `"all day"` |

**"Overnight" start-time logic:** if `now` is before 10 pm, `startTime` is
tonight at 10 pm; otherwise `startTime` is `now` (the user is already in the
overnight window). `endTime` is always 8 am on the next calendar day after
`startTime`.

**"Until" future-time logic:** compute the target time on today's calendar date.
If the computed target is ≤ `now`, advance it by 24 hours (wrap to tomorrow).

**Numeric parsing:** `N` must be a positive integer (1–999). Reject decimals and
zero.

**Unrecognized text:** return `null`. Do not attempt partial matching or
fallback guesses.

### Module-level helpers (private)

```typescript
function setTimeOfDay(base: Date, hour: number, minute: number): Date
// Returns a new Date with the same calendar date as `base` but hour/minute set
// in local time, seconds/ms cleared.

function tomorrowAt(base: Date, hour: number, minute: number): Date
// Returns a new Date for the NEXT calendar day after `base` at hour:minute
// local time.
```

### Full implementation contract

```typescript
export function parseQuery(text: string, now: Date): ParkingQuery | null {
  const t = text.trim().toLowerCase();

  // overnight / all night / tonight
  if (t === "overnight" || t === "all night" || t === "tonight") {
    const OVERNIGHT_START_HOUR = 22; // 10 pm
    const OVERNIGHT_END_HOUR = 8;
    const tenPmToday = setTimeOfDay(now, OVERNIGHT_START_HOUR, 0);
    const startTime = now.getTime() < tenPmToday.getTime() ? tenPmToday : new Date(now);
    const endTime = tomorrowAt(startTime, OVERNIGHT_END_HOUR, 0);
    return { startTime, endTime, label: "overnight" };
  }

  // right now / now
  if (t === "right now" || t === "now") {
    return {
      startTime: new Date(now),
      endTime: new Date(now.getTime() + 60 * 60 * 1000),
      label: "right now",
    };
  }

  // all day
  if (t === "all day") {
    const startTime = setTimeOfDay(now, 0, 0);
    const endTime = setTimeOfDay(now, 23, 59);
    return { startTime, endTime, label: "all day" };
  }

  // for N hours / for N hour
  const hoursMatch = t.match(/^for\s+(\d+)\s+hours?$/);
  if (hoursMatch !== null) {
    const n = parseInt(hoursMatch[1] ?? "0", 10);
    if (n <= 0 || n > 999) return null;
    return {
      startTime: new Date(now),
      endTime: new Date(now.getTime() + n * 60 * 60 * 1000),
      label: `for ${n} hour${n === 1 ? "" : "s"}`,
    };
  }

  // for N minutes / mins / min
  const minutesMatch = t.match(/^for\s+(\d+)\s+(?:minutes?|mins?)$/);
  if (minutesMatch !== null) {
    const n = parseInt(minutesMatch[1] ?? "0", 10);
    if (n <= 0 || n > 999) return null;
    return {
      startTime: new Date(now),
      endTime: new Date(now.getTime() + n * 60 * 1000),
      label: `for ${n} minute${n === 1 ? "" : "s"}`,
    };
  }

  // until X am/pm  or  until H:MM am/pm
  const untilMatch = t.match(/^until\s+(\d{1,2})(?::(\d{2}))?\s*(am|pm)$/);
  if (untilMatch !== null) {
    let hour = parseInt(untilMatch[1] ?? "0", 10);
    const minute = parseInt(untilMatch[2] ?? "0", 10);
    const period = untilMatch[3];
    if (hour < 1 || hour > 12 || minute > 59) return null;
    if (period === "pm" && hour !== 12) hour += 12;
    if (period === "am" && hour === 12) hour = 0;
    let endTime = setTimeOfDay(now, hour, minute);
    if (endTime.getTime() <= now.getTime()) {
      endTime = new Date(endTime.getTime() + 24 * 60 * 60 * 1000);
    }
    const minuteStr = untilMatch[2] !== undefined ? `:${untilMatch[2]}` : "";
    const label = `until ${untilMatch[1]}${minuteStr} ${period ?? ""}`.trim();
    return { startTime: new Date(now), endTime, label };
  }

  return null;
}
```

## Test list

1. GIVEN text `"overnight"` and `now` before 10 pm,
   WHEN `parseQuery` called,
   THEN `startTime` is tonight at 10 pm, `endTime` is tomorrow at 8 am, `label` is `"overnight"`

2. GIVEN text `"overnight"` and `now` is 11 pm,
   WHEN `parseQuery` called,
   THEN `startTime` equals `now`, `endTime` is tomorrow at 8 am

3. GIVEN text `"all night"`,
   WHEN `parseQuery` called,
   THEN result equals the `"overnight"` result for the same `now`

4. GIVEN text `"tonight"`,
   WHEN `parseQuery` called,
   THEN result equals the `"overnight"` result for the same `now`

5. GIVEN text `"right now"`,
   WHEN `parseQuery` called,
   THEN `startTime` equals `now`, `endTime` is `now + 1 hour`

6. GIVEN text `"now"`,
   WHEN `parseQuery` called,
   THEN `startTime` equals `now`, `endTime` is `now + 1 hour`

7. GIVEN text `"for 2 hours"`,
   WHEN `parseQuery` called,
   THEN `endTime` is `now + 2 hours`, `label` is `"for 2 hours"`

8. GIVEN text `"for 1 hour"`,
   WHEN `parseQuery` called,
   THEN `label` is `"for 1 hour"` (singular)

9. GIVEN text `"for 30 minutes"`,
   WHEN `parseQuery` called,
   THEN `endTime` is `now + 30 minutes`, `label` is `"for 30 minutes"`

10. GIVEN text `"for 1 min"`,
    WHEN `parseQuery` called,
    THEN `endTime` is `now + 1 minute`, `label` is `"for 1 minute"` (singular)

11. GIVEN text `"until 8am"` and `now` is before 8 am today,
    WHEN `parseQuery` called,
    THEN `endTime` is today at 8:00 am

12. GIVEN text `"until 8am"` and `now` is after 8 am today,
    WHEN `parseQuery` called,
    THEN `endTime` is tomorrow at 8:00 am

13. GIVEN text `"until 10:30 pm"`,
    WHEN `parseQuery` called,
    THEN `endTime` has correct hour (22) and minute (30)

14. GIVEN text `"all day"`,
    WHEN `parseQuery` called,
    THEN `startTime` is midnight today, `endTime` is 11:59 pm today

15. GIVEN text `"park here for a while"` (unrecognized),
    WHEN `parseQuery` called,
    THEN returns `null`

16. GIVEN text `"for 0 hours"`,
    WHEN `parseQuery` called,
    THEN returns `null`

17. GIVEN text `"FOR 3 HOURS"` (uppercase),
    WHEN `parseQuery` called,
    THEN parses correctly (case-insensitive)
