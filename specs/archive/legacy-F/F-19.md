## F-19 Tow Zones Legend Toggle `[TODO]`

Adds a fixed map overlay control in the top-left corner, just below the Leaflet zoom buttons, that lets the user show or hide all tow zone markers. The control displays "Tow Zones" as the title with "Enabled" or "Disabled" centered below it as a live status indicator, and a small red dot icon matching the tow sign markers.

This feature modifies `app/index.html`, `app/style.css`, `app/map.ts`, and `app/main.ts`.

---

**`app/map.ts`** — add one new export:

```typescript
export function setTowSignsVisible(visible: boolean): void
```

### F-19.3 `setTowSignsVisible` (`app/map.ts`)

Add after `clearSpotMarker` and before `showStreetPopup`:

```typescript
export function setTowSignsVisible(visible: boolean): void {
  const mapEl = document.getElementById("map");
  if (mapEl === null) return;
  if (visible) {
    mapEl.classList.remove("tow-signs-hidden");
  } else {
    mapEl.classList.add("tow-signs-hidden");
  }
}
```

**Tests**
- GIVEN a DOM with `<div id="map"></div>`, WHEN `setTowSignsVisible(false)` is called, THEN `#map` has class `tow-signs-hidden`.
- GIVEN a DOM with `<div id="map" class="tow-signs-hidden"></div>`, WHEN `setTowSignsVisible(true)` is called, THEN `#map` does not have class `tow-signs-hidden`.
- GIVEN no `#map` element in the DOM, WHEN `setTowSignsVisible(false)` is called, THEN no error is thrown.

### F-19.4 Toggle wiring (`app/main.ts`)

In `initBrowserApp`, import `setTowSignsVisible` from `./map` and wire the button after map initialization:

```typescript
const towLegend = document.getElementById("tow-legend");
const towToggle = document.getElementById("tow-toggle");
if (towLegend !== null && towToggle !== null) {
  const towStatus = towToggle.querySelector<HTMLElement>(".tow-status");
  towToggle.addEventListener("click", () => {
    const isOn = !towLegend.classList.contains("tow-off");
    setTowSignsVisible(!isOn);
    towLegend.classList.toggle("tow-off", isOn);
    towToggle.setAttribute("aria-pressed", String(!isOn));
    if (towStatus !== null) {
      towStatus.textContent = isOn ? "Disabled" : "Enabled";
    }
  });
}
```

**Tests**
- GIVEN the toggle is clicked once (initially enabled), THEN `#tow-legend` has class `tow-off` and `.tow-status` text is `"Disabled"`.
- GIVEN the toggle is clicked twice (enabled → disabled → enabled), THEN `#tow-legend` does not have class `tow-off` and `.tow-status` text is `"Enabled"`.
- GIVEN the toggle is clicked, THEN `aria-pressed` on `#tow-toggle` reflects the new state (`"false"` when disabled, `"true"` when enabled).
