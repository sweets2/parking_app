# Spec Archive

This directory contains historical feature specs that are no longer referenced
by the active harness graph.

Active specs are:

- `specs/v2/CF-*.md` for the current foundation features.
- `specs/F-46A.md` through `specs/F-58.md` for the current Check | Rules
  feature sequence still referenced by `harness/features.json`.

Archived specs are retained for traceability only. Do not use archived specs as
implementation authority unless `harness/features.json` is explicitly updated to
point back to them.

## Folders

- `legacy-F/` — old root `F-*` and discovery specs not referenced by the current
  graph.
- `generated-app-spec-copies/` — old copied specs that previously lived under
  `generated_app/specs/`.
