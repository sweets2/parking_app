## Hard constraints

- No source files beyond the three listed above.
- `npm install` is run as `post_build_command` — do not run it during spec implementation.
- `strict: true` is mandatory in tsconfig.
- `environment: "node"` is mandatory in vitest.config.ts.
- TypeScript `strict: true` always on. No `as any` casts. No `!` non-null assertions. All exported symbols use named exports — no default exports.
- Never run `npm run build` during implementation (build emits app.js/sw.js which shadow TypeScript sources in vitest).
- Tests run in Node (`environment: "node"` in vitest.config). No browser globals available at test time.
