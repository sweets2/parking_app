# CF-13 — PWA Shell

## What this is

Progressive Web App configuration: the web app manifest and service worker.

## Output files

- `app/manifest.json`
- `app/sw.ts`

## `app/manifest.json`

```json
{
  "name": "Hoboken Parking",
  "short_name": "HP Parking",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#ffffff",
  "theme_color": "#ffffff",
  "icons": [
    {
      "src": "icon-192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "icon-512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}
```

## `app/sw.ts`

```typescript
const CACHE_NAME = "hoboken-parking-v2";

const APP_SHELL_URLS = [
  "/",
  "/index.html",
  "/app.js",
  "/style.css",
  "/manifest.json",
  "https://unpkg.com/leaflet@1.9.4/dist/leaflet.js",
  "https://unpkg.com/leaflet@1.9.4/dist/leaflet.css",
];
```

### Strategy

- **Install:** cache all `APP_SHELL_URLS` and call `skipWaiting()`.
- **Activate:** delete all caches with names not equal to `CACHE_NAME`.
- **Fetch:**
  - For `data/latest.json`: network-first (try network, fall back to cache on failure).
  - For any URL in `APP_SHELL_URLS` (or matching the app origin): cache-first (return from cache if available, otherwise fetch and cache).
  - All other requests: pass through to network.

### Compilation

`app/sw.ts` is compiled by esbuild as part of the build script:

```
esbuild app/sw.ts --bundle --outfile=app/sw.js --platform=browser
```

This step is already included in the `build` script in `package.json` (CF-00).

## Hard constraints

- No `as any`. No `!`. Named exports are not needed (service workers have no exports — the file uses `self.addEventListener`).
- `CACHE_NAME` must be `"hoboken-parking-v2"` exactly.
- `skipWaiting()` must be called on install.
- `data/latest.json` must use network-first strategy.
- `app/sw.ts` is compiled to `app/sw.js` by esbuild — do NOT write `app/sw.js` directly; write the TypeScript source.
- No tests for the service worker (it runs in a service worker context, not Node).
